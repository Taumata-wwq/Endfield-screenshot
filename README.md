# 终末地俯瞰模式截图工具

一个用于《明日方舟：终末地》的自动截图和拼接工具，可以在俯瞰模式下自动捕捉游戏画面并拼接成完整的蓝图。

**当前版本：v0.2.0**

## 功能特点

- 提供图形界面
- 自动识别游戏窗口后网格化截图，支持预设区域或自定义行列数
- 蛇形遍历路径，自动处理重叠区域拼接，自动保存拼接后的结果
- 随机延迟模拟
- 快捷键控制

## 环境要求

- Python 3.8+
- Windows 操作系统

## 安装依赖

```bash
pip install pyautogui pygetwindow pillow keyboard
```

## 使用方法

### 方式一：直接运行 exe（推荐）

在 `Releases` 下找到 `终末地截图工具.exe`，保存本地运行即可。

### 方式二：从源码运行

运行程序：

```bash
python auto_screenshot.py
```

### 准备工作

1. 打开《明日方舟：终末地》游戏，设置为 **1280×720 窗口模式**
2. 进入俯瞰模式的批量选择状态
3. 若使用预设区域截图，须将画面缩至最小后，将画面移动到**最左上角**

### 快捷键

| 热键   | 功能           |
| ------ | -------------- |
| `F1`   | 开始自动截图   |
| `F2`   | 停止截图       |
| `F3`   | 手动截图一次   |
| `F4`  | 退出程序       |

### 选择截图区域

在界面下拉框中选择预设区域，或选择"自定义"后手动输入行列数。

**预设区域：**

| 区域名称       | 网格大小 |
| -------------- | -------- |
| 武陵-武陵城    | 14×8     |
| 武陵-景玉谷    | 10×6     |
| 谷地-枢纽区    | 13×7     |
| 谷地-谷地通道  | 9×5      |
| 谷地-供能高地  | 9×5      |
| 谷地-源石研究园| 9×5      |

## 工作原理

1. **窗口识别**：自动查找标题包含 "Endfield" 的窗口
2. **蛇形遍历**：从左上角开始，按行蛇形移动相机（左→右，下一行右→左）
3. **相机控制**：通过模拟鼠标中键拖拽移动游戏视角
4. **图片拼接**：根据设定的重叠率自动拼接所有截图

## 配置说明

程序默认配置位于 `CONFIG` 字典：

| 参数                | 默认值  | 说明                       |
| ------------------- | ------- | -------------------------- |
| `stabilize_delay`   | 0.08秒  | 拖拽后等待画面稳定的时间   |
| `screenshot_delay`  | 0.08秒  | 截图前的等待时间           |
| `capture_region`    | (0.62, 0.62) | 截图区域宽高占窗口比例 |
| `overlap`           | (0.005, 0.010) | 水平/垂直拼接重叠率  |
| `drag_margin`       | 40像素  | 拖拽操作距离窗口边缘的距离 |
| `drag_duration`     | 0.08秒  | 拖拽动作持续时间           |

> 所有时间参数均带有 ±10% 的随机波动模拟人工

## 输出

截图保存在 `screenshots/` 目录：

| 类型       | 文件名格式                        |
| ---------- | --------------------------------- |
| 自动拼接   | `stitched_YYYYMMDD_HHMMSS.jpg`    |
| 手动截图   | `manual_YYYYMMDD_HHMMSS.jpg`      |

## 注意事项

1. 程序自动获取管理员权限用于模拟鼠标操作，需确认 UAC 提示
2. 截图过程中请勿移动游戏窗口及鼠标，确保游戏窗口处于前台且未被遮挡

## 打包说明

使用 PyInstaller 打包为独立 exe：

```bash
pyinstaller 终末地截图工具.spec --clean
```


## 技术栈

- **tkinter** - 图形用户界面
- **pyautogui** - 屏幕截图、鼠标操作
- **pygetwindow** - 游戏窗口管理
- **Pillow** - 图片拼接处理
- **keyboard** - 全局热键监听
- **PyInstaller** - 打包为独立可执行文件
