# 终末地俯瞰模式截图工具

一个用于《明日方舟：终末地》的自动截图和拼接工具，可以在俯瞰模式下自动捕捉游戏画面并拼接成完整的蓝图。

## 功能特点

- 自动识别游戏窗口、网格截图，支持自定义行列数，自动处理重叠区域
- 提供快捷键控制
- 预设截图配置
- 自动保存拼接后的结果

## 环境要求

- Python 3.8+
- Windows 操作系统

## 安装依赖

```bash
pip install pyautogui pygetwindow pillow keyboard
```

## 使用方法

### 1. 启动程序

以**管理员权限**运行程序：

```bash
python auto_screenshot.py
```

### 2. 准备工作

1. 打开《明日方舟：终末地》游戏设置为[1280*720窗口]
2. 进入俯瞰模式的批量选择状态
3. 将画面缩放至最小（无法继续放大）
4. 将画面移动到最左上角

### 3. 快捷键

| 热键   | 功能         |
| ------ | ------------ |
| `F1` | 开始自动截图 |
| `F2` | 停止截图     |
| `F3` | 手动截图一次 |
| `F4` | 退出程序     |

### 4. 选择截图区域

按下 `F1` 后，可以选择预设区域或自定义：

**预设区域：**

- `1` - 武陵 - 武陵城 (河流左侧或右侧) - 14×8
- `2` - 武陵 - 景玉谷 (上方或下方水潭) - 10×6
- `3` - 谷地 - 枢纽区 (存储线右下或左上) - 14×7
- `4` - 谷地 - 谷地通道 (存储线上或下) - 9×5
- `5` - 谷地 - 供能高地 (存储线上或下) - 9×5
- `6` - 谷地 - 源石研究园 (存储线上或下) - 9×5

**自定义：** 输入 `0` 可自定义行列数，自行截取需要的部分

## 配置说明

程序默认配置位于 `DEFAULT_CONFIG`：

- `grid_size`: 网格大小（行列数）
- `capture_region_x/y`: 截图区域比例
- `overlap_x/y`: 拼接重叠率
- `drag_margin`: 拖拽边距
- `drag_duration`: 拖拽持续时间
- `output_folder`: 输出文件夹路径

## 输出

- 单张截图保存在 `screenshots/` 目录
- 拼接完成的大图保存在 `screenshots/stitched_YYYYMMDD_HHMMSS.jpg`

## 注意事项

1. 程序需要**管理员权限**运行（用于模拟鼠标操作）
2. 截图过程中请勿移动游戏窗口及鼠标

## 技术栈

- Python 3.x
- pyautogui - 自动化控制
- pygetwindow - 窗口管理
- Pillow (PIL) - 图片处理
- keyboard - 热键监听
